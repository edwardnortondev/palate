## What is Palate?

Palate is a personal web app for archiving the things that resonate with you aesthetically. It is not a mood board, not a social bookmarking tool, and not Pinterest. It is a private record — a running log of taste.

You save an item (an image, a film, a quote, a place) and tag it. Over time the tags accumulate into a picture of what you actually respond to, stripped of performance and audience.

### The problem it solves

Most tools for saving things are either social (Pinterest, Are.na) or chaotic (browser bookmarks, screenshot folders). Palate is neither. It is structured enough to be searchable, private enough to be honest, and simple enough to stay out of the way.

### What you can save

| Type      | Examples                               |
| --------- | -------------------------------------- |
| `image`   | Photography, art, design, architecture |
| `link`    | Articles, essays, websites             |
| `quote`   | Sentences worth keeping                |
| `film`    | Movies, short films, scenes            |
| `music`   | Tracks, albums, artists                |
| `place`   | Locations, spaces, cities              |
| `fashion` | Garments, designers, looks             |

### Philosophy

The design follows the same principle as the content it holds: nothing unnecessary. One user, one archive, no noise.
